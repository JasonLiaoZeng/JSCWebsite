To Do:

1) Head to Unity Downlad Page

2) Choose the hardware you are using(Window or Mac OS), then accept terms and download Unity Personal. Unity Personal is a free version of Unity that we can use legally for internal project until we reaches a yearly revenue of 100K.

3) After download finish, open UnityHubSetUp.exe to proceed to installation. Accept all terms and default setup, then proceed until installation completes.

4) The default setting should open Unity Hub for you. Otherwise, open Unity Hub from your desktop or app folder, then create an Unity ID and sign in on your Unity Hub with the Unity ID you had just created. Do not worry if your interface has a different color, it is because we are on different theme and it does not affect the functionality. If you prefer the dark theme, you can set it on Preferences -> Apperarence -> Theme -> Dark.
