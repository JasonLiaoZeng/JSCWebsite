Project 1: Tik-Tak-Toe!

As mentioned, codes are the essential part of any software. In this project, students will be utilizing what they had learned in lecture to build the simple simple game Tik-Tac-Toe. Part of this project is moving around and getting farmiliar with Unity Game Engine interface and building tools, and the other part is actually writing the code.
